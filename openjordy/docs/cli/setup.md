# `openjordy setup`

Initialize `~/.openjordy/openjordy.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
openjordy setup
openjordy setup --workspace ~/.openjordy/workspace
```

To run the wizard via setup:

```bash
openjordy setup --wizard
```
